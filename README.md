# wheatherApi
wheather basedEvent Api Documentation 🚀 Tech Stack Node -Backend

Backend

Node.js
Express.js
MongoDB Atlas (Mongoose ORM)


Dotenv (environment variables) ⚙️ Setup Instructions
Clone Repository https://github.com/NEHA-DEV-FULLSTACK/wheatherApi cd weatherApi

Setup Backend cd nodeBackend npm install

Create a .env file with:
MONGO_URI= mongodb://localhost:27017/WeathertestAPI
PORT=3000

Start backend: npm run dev 
 git config --global user.name "Your Name"
    git config --global user.email "you@example.com"

Api - 
1) user signup-to create user using usename and password
http://localhost:3000/api/user/login

2) user login using usename and password and get token
http://localhost:3000/api/user/login

3) logout user  
http://localhost:3000/api/logout


----event api ------
4) create whether event http://localhost:3000/api/event/

5)get event by id -get http://localhost:3000/api/event/68b025925e9674b7701465da
6) update event - put request http://localhost:3000/api/event/68b021a30b8ffec7e9da775c

7)delete event by id -delete http://localhost:3000/api/event/68b021a30b8ffec7e9da775c
